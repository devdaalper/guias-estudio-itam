Función, dominio e imagen

Definición

Función de varias variables

Enunciado formal Una función de n variables f : D → R, con D ⊆ Rn, asigna a cada punto (x₁, …, xn) ∈ D un único número real f(x₁, …, xn). El conjunto D es el dominio. Para n = 2 se escribe z = f(x, y): a cada pareja (x, y) del plano le corresponde una sola altura z.
Por qué

La palabra clave es único: si una entrada pudiera producir dos salidas, no sería función. La entrada es un punto de Rn (una pareja cuando n = 2); la salida es siempre un solo número real (por eso es «con valores en R»).

Recall · enuncia de memoria

Define f : D → R con D ⊆ Rn. ¿Qué es la entrada, qué es la salida y qué palabra no puede faltar?

Respuesta
f : D → R, D ⊆ Rn, asigna a cada punto (x₁, …, xn) del dominio un único real. Entrada = punto de Rn; salida = un número.
¿Cómo te salió?
Definición

Dominio, imagen y gráfica

Enunciado formal El dominio de f es el conjunto de puntos donde la fórmula tiene sentido: D = { (x, y) : f(x, y) está definida } ⊆ R2. La imagen (o rango) es el conjunto de valores de salida: Im(f) = { f(x, y) : (x, y) ∈ D } ⊆ R. La gráfica de z = f(x, y) es la superficie { (x, y, z) : z = f(x, y), (x, y) ∈ D } ⊆ R3.
Por qué

Son tres conjuntos que viven en dimensiones distintas: el dominio en el plano (R2), la imagen en la recta (R), y la gráfica en el espacio (R3). Confundirlos es el error clásico: la imagen NO es un conjunto de parejas, son las alturas.

Recall · enuncia de memoria

Define dominio, imagen y gráfica de z = f(x, y). ¿En qué espacio vive cada uno?

Respuesta
Dominio = donde f está definida (⊆ R2). Imagen = valores de salida (⊆ R). Gráfica = { (x, y, z) : z = f(x, y) } (⊆ R3).
¿Cómo te salió?
Regla

Las 3 restricciones del dominio

Enunciado formal Solo tres operaciones pueden dejar una fórmula sin definir; el dominio es todo R2 menos lo que ellas prohíben:
  • 1División: el denominador debe ser distinto de cero (≠ 0).
  • 2Raíz de índice par (√ , raíz cuarta …): el radicando debe ser mayor o igual que cero (≥ 0).
  • 3Logaritmo: el argumento debe ser estrictamente positivo (> 0).
Por qué

Son las mismas tres de una variable, y ninguna otra cosa restringe. El signo de x, de y o del resultado no prohíbe nada por sí solo. Ojo con el ≥ de la raíz frente al > del logaritmo: la raíz sí admite el 0 (√0 = 0), el logaritmo no (ln 0 no existe).

Recall · enuncia de memoria

Enuncia las 3 restricciones del dominio. Cuida el símbolo exacto: ≠ 0, ≥ 0 o > 0 en cada una.

Respuesta
Denominador ≠ 0 · radicando de raíz par ≥ 0 · argumento de logaritmo > 0. Nada más restringe.
¿Cómo te salió?

Conjuntos de nivel

Definición

Conjunto, curva y superficie de nivel

Enunciado formal El conjunto de nivel k de f es el conjunto de puntos del dominio donde la función vale la constante k: { P ∈ D : f(P) = k }. Se llama curva de nivel cuando f tiene dos variables (vive en el plano) y superficie de nivel cuando tiene tres (vive en el espacio). Se obtiene igualando la fórmula a la constante: f(x, y) = k.
Por qué

Es la idea del mapa topográfico: cada línea une los puntos a la misma altura. El nivel vive en el dominio (el plano para dos variables), no en la gráfica 3D. Si ningún punto satisface f = k, el conjunto de nivel es vacío (∅).

Recall · enuncia de memoria

Define conjunto de nivel k. ¿Cuándo es «curva» y cuándo «superficie»? ¿En qué espacio vive?

Respuesta
{ P : f(P) = k } = igualar la fórmula a la constante. Curva de nivel si 2 variables (en el plano), superficie de nivel si 3. Vacío si nadie cumple f = k.
¿Cómo te salió?
Caso clave

Curvas de nivel de x² + y²

Enunciado formal Para f(x, y) = x² + y², la curva de nivel k es x² + y² = k:
  • aSi k > 0: un círculo centrado en el origen de radio √k.
  • bSi k = 0: un solo punto, el origen (0, 0).
  • cSi k < 0: el conjunto vacío (∅), porque una suma de cuadrados nunca es negativa.
Por qué

x² + y² es la distancia al origen al cuadrado, y una distancia nunca es negativa: por eso k < 0 no tiene solución. Este patrón cae siempre: reconocer x² + y² = k como círculo de radio √k ahorra media página de cuentas.

Recall · enuncia de memoria

Describe la curva de nivel k de x² + y² en los tres casos: k > 0, k = 0 y k < 0.

Respuesta
k > 0 → círculo de radio √k. k = 0 → el origen (un punto). k < 0 → vacío (suma de cuadrados no es negativa).
¿Cómo te salió?

Límite doble

Definición

Límite de una función de dos variables (ε–δ)

Enunciado formal lím(x,y)→(a,b) f(x, y) = L significa: para todo ε > 0 existe δ > 0 tal que, si 0 < √((x − a)² + (y − b)²) < δ, entonces | f(x, y) − L | < ε. Es decir: acercándose lo suficiente al punto (por cualquier lado), la función queda tan cerca de L como se quiera.
Por qué

La diferencia con una variable está en el «acercarse»: aquí la distancia al punto es la del plano, √((x−a)²+(y−b)²), y ese entorno es un disco completo. Por eso el acercamiento no es por izquierda/derecha, sino por todas las direcciones a la vez. El 0 < … excluye el punto mismo: el valor en (a, b) no importa para el límite.

Recall · enuncia de memoria

Enuncia la definición ε–δ del límite doble. ¿Qué papel juega la distancia del plano y por qué se pide 0 < distancia?

Respuesta
∀ ε > 0 ∃ δ > 0: si 0 < √((x−a)²+(y−b)²) < δ entonces | f − L | < ε. El entorno es un disco (todas las direcciones); 0 < excluye el punto, así que f(a, b) no cuenta.
¿Cómo te salió?
Teorema

Unicidad del límite

Enunciado formal Si el límite lím(x,y)→(a,b) f(x, y) existe, entonces es único: por toda trayectoria que se acerque a (a, b) dentro del dominio, f debe tender al mismo valor L.
Por qué

Es el teorema que legitima el método de los caminos. Como el límite (si existe) tiene que ser el mismo por todas las direcciones, basta hallar dos trayectorias con valores distintos para concluir que no existe. La contradicción con la unicidad es lo que cierra la prueba.

Recall · enuncia de memoria

Enuncia la unicidad del límite doble. ¿Qué método adversarial habilita este teorema?

Respuesta
Si el límite existe, es único: por toda trayectoria f tiende al mismo L. Habilita negar el límite mostrando dos caminos con valores distintos.
¿Cómo te salió?
Criterio

Prueba de los dos caminos (negar el límite)

Enunciado formal Si existen dos trayectorias C₁ y C₂ que llegan a (a, b) tales que límC₁ f = L₁, límC₂ f = L₂, con L₁ ≠ L₂, entonces lím(x,y)→(a,b) f(x, y) no existe. Caminos típicos: los ejes (y = 0, x = 0) y las rectas y = mx (o parábolas y = mx²).
Por qué

Es la unicidad usada al revés: dos valores distintos por dos caminos contradicen que haya un solo límite. Atención al alcance: este criterio solo sirve para negar. Que muchos caminos coincidan NO prueba que el límite exista; para afirmarlo hace falta polares o el sándwich.

Recall · enuncia de memoria

Enuncia el criterio de los dos caminos. ¿Qué basta encontrar y para qué sirve (y para qué NO sirve)?

Respuesta
Dos caminos con límites L₁ ≠ L₂ ⇒ el límite no existe. Solo sirve para NEGAR: que muchos caminos coincidan no prueba que exista.
¿Cómo te salió?
Método

Cambio a coordenadas polares

Enunciado formal Para un límite en el origen se sustituye x = r·cos θ, y = r·sen θ, con lo que x² + y² = r². Acercarse a (0, 0) equivale a hacer r → 0. Si al simplificar queda f = rp·g(θ) con p > 0 y g(θ) acotada (| g(θ) | ≤ M para todo θ), entonces el límite existe y vale 0, sin importar el ángulo.
Por qué

El truco funciona porque x² + y² = r² colapsa la distancia en una sola variable, r. Si sobrevive un factor rp (p > 0) multiplicando algo acotado, rp → 0 arrastra todo a 0. Clave: el resultado debe quedar independiente de θ; si el ángulo no se va, el límite podría no existir.

Recall · enuncia de memoria

Escribe el cambio a polares y la condición para concluir que el límite en el origen es 0. ¿Qué patrón lo dispara?

Respuesta
x = r·cos θ, y = r·sen θ ⇒ x² + y² = r²; r → 0. Si queda rp·g(θ) con p > 0 y g acotada, el límite es 0, independiente de θ. Lo dispara ver x² + y².
¿Cómo te salió?
Teorema

Teorema del sándwich (acotamiento)

Enunciado formal Si en un entorno del punto se cumple g(x, y) ≤ f(x, y) ≤ h(x, y) y además lím g = lím h = L (los dos extremos tienden al mismo L), entonces lím f = L. Versión útil: si | f(x, y) − L | ≤ φ(x, y) y φ → 0, entonces lím f = L.
Por qué

Si f queda atrapada entre dos funciones que van al mismo sitio, no le queda otra que ir ahí también. En polares es el remate: cuando f = rp·g(θ) con | g | ≤ M, se tiene | f | ≤ M·rp, y como M·rp → 0, el sándwich fuerza el límite a 0.

Recall · enuncia de memoria

Enuncia el teorema del sándwich para límites dobles y su versión con | f − L | ≤ φ. ¿Cómo remata en polares?

Respuesta
g ≤ f ≤ h con lím g = lím h = L ⇒ lím f = L. O bien | f − L | ≤ φ y φ → 0 ⇒ lím f = L. En polares: | f | ≤ M·rp → 0.
¿Cómo te salió?

Continuidad

Definición

Continuidad en un punto

Enunciado formal f es continua en (a, b) si se cumplen las tres condiciones:
  • 1f(a, b) existe (el punto está en el dominio).
  • 2lím(x,y)→(a,b) f(x, y) existe.
  • 3Los dos coinciden: lím(x,y)→(a,b) f(x, y) = f(a, b).
De forma compacta: lím(x,y)→(a,b) f(x, y) = f(a, b).
Por qué

La condición 3 es la que se olvida: no basta que existan valor y límite, tienen que ser iguales. Si el límite es 0 pero a la función la definieron con 5 en el punto, hay un salto: es discontinua. En funciones a trozos, el origen es justo donde hay que revisar las tres.

Recall · enuncia de memoria

Enuncia las 3 condiciones de continuidad en (a, b). ¿Cuál se olvida más y qué pasa si falla?

Respuesta
(1) f(a, b) existe · (2) el límite existe · (3) son iguales: lím f = f(a, b). Se olvida la 3; si falla, hay un salto (discontinua).
¿Cómo te salió?
Teorema

Álgebra y composición de funciones continuas

Enunciado formal Si f y g son continuas en (a, b), también lo son f + g, f − g, f·g y f ⁄ g (esta última donde g(a, b) ≠ 0). Además, la composición de continuas es continua: si f es continua en (a, b) y una función h de una variable es continua en f(a, b), entonces h∘f es continua en (a, b). Consecuencia: polinomios, y las funciones elementales (√, ln, sen, exp …) compuestas con ellos, son continuas en todo su dominio.
Por qué

Este teorema es lo que evita usar ε–δ cada vez: como los polinomios son continuos y la continuidad se hereda por suma, producto, cociente y composición, casi toda función «de fórmula» es continua salvo donde se rompe (denominador 0, radicando negativo, etc.). Los problemas interesantes de límite viven justo en esos puntos frontera.

Recall · enuncia de memoria

¿Qué operaciones preservan la continuidad? Enuncia también la composición y la consecuencia sobre polinomios.

Respuesta
Suma, resta, producto y cociente (con denominador ≠ 0) de continuas son continuas; la composición de continuas es continua. Por eso polinomios y elementales son continuos en su dominio.
¿Cómo te salió?

La ficha de memoria

En el examen no hay formulario. Esto es lo que debes poder escribir de memoria, palabra por palabra.

  • ✓ Función f : D → R, D ⊆ Rn: a cada punto un único real. Entrada = punto de Rn; salida = un número.
  • ✓ Dominio ⊆ R2 (donde se define) · imagen ⊆ R (valores de salida) · gráfica ⊆ R3.
  • ✓ 3 restricciones: denominador ≠ 0 · radicando de raíz par ≥ 0 · argumento de logaritmo > 0.
  • ✓ Conjunto de nivel k = { f = k }; curva (2 var, en el plano) o superficie (3 var). Vacío si nadie cumple.
  • ✓ x² + y² = k: círculo de radio √k si k > 0 · el origen si k = 0 · vacío si k < 0.
  • ✓ Límite ε–δ: ∀ ε > 0 ∃ δ > 0 con 0 < √((x−a)²+(y−b)²) < δ ⇒ | f − L | < ε. Entorno = disco.
  • ✓ Unicidad: si el límite existe, es el mismo por toda trayectoria.
  • ✓ Dos caminos con L₁ ≠ L₂ ⇒ el límite NO existe. Solo sirve para negar, nunca para afirmar.
  • ✓ Polares: x = r·cos θ, y = r·sen θ ⇒ x² + y² = r²; r → 0. Si queda rp·g(θ) con g acotada, límite = 0.
  • ✓ Sándwich: g ≤ f ≤ h con lím g = lím h = L ⇒ lím f = L. O | f − L | ≤ φ → 0 ⇒ lím f = L.
  • ✓ Continua en (a, b): existe f(a, b) · existe el límite · son iguales (lím f = f(a, b)).
  • ✓ Suma, producto, cociente (denom ≠ 0) y composición de continuas son continuas; polinomios son continuos.
