Subespacio

Definición

Subespacio de Rⁿ

Enunciado formal Un subconjunto S ⊆ Rⁿ es un subespacio de Rⁿ si y solo si cumple las tres condiciones:
  • 1El vector cero pertenece a S: 0 ∈ S.
  • 2S es cerrado bajo la suma: si u ∈ S y v ∈ S, entonces u + v ∈ S.
  • 3S es cerrado bajo producto por escalar: si u ∈ S y c ∈ R, entonces c·u ∈ S.
Por qué

Las condiciones 2 y 3 dicen que S es cerrado bajo combinaciones lineales. La condición 1 no es redundante como suele creerse: descarta el conjunto vacío y sirve como test rápido de descarte. De hecho, si S ≠ ∅ y es cerrado bajo escalar, tomar c = 0 ya obliga a 0 ∈ S; por eso muchos textos piden 1 explícitamente para no depender de que S sea no vacío.

Recall · enuncia de memoria

Enuncia las 3 condiciones exactas para que S ⊆ Rⁿ sea subespacio. Dilas antes de revelar.

Respuesta
(1) 0 ∈ S. (2) u, v ∈ S ⇒ u + v ∈ S. (3) u ∈ S, c ∈ R ⇒ c·u ∈ S.
¿Cómo te salió?

Combinación lineal y generado

Definición

Combinación lineal y espacio generado

Enunciado formal Sean v₁, …, vk ∈ Rⁿ. Una combinación lineal de ellos es cualquier vector de la forma c₁·v₁ + c₂·v₂ + … + ck·vk, con c₁, …, ck ∈ R (escalares). El generado (span) es el conjunto de todas sus combinaciones lineales: gen{v₁, …, vk} = { c₁·v₁ + … + ck·vk : c₁, …, ck ∈ R }.
Por qué

El generado es «todo lo alcanzable» combinando esos vectores. Preguntar si w ∈ gen{v₁, …, vk} equivale a preguntar si el sistema con columnas v₁, …, vk y término independiente w es consistente.

Recall · enuncia de memoria

Define con precisión gen{v₁, …, vk}. Usa notación de conjunto.

Respuesta
gen{v₁, …, vk} = { c₁·v₁ + … + ck·vk : c₁, …, ck ∈ R }, el conjunto de todas sus combinaciones lineales.
¿Cómo te salió?
Teorema

El generado es un subespacio

Enunciado formal Para cualesquiera v₁, …, vk ∈ Rⁿ, el conjunto gen{v₁, …, vk} es un subespacio de Rⁿ. Más aún, es el menor subespacio que contiene a v₁, …, vk.
Por qué

Contiene al 0 (todos los escalares en cero). La suma de dos combinaciones lineales es otra combinación lineal, y un escalar por una combinación lineal vuelve a ser combinación lineal: cierra bajo suma y bajo escalar. Cumple las 3 condiciones, luego es subespacio.

Recall · enuncia de memoria

Enuncia el teorema sobre el generado y da la idea de su demostración (tres cierres).

Respuesta
gen{v₁, …, vk} es un subespacio (el menor que contiene a esos vectores): tiene el 0, y suma y escalar de combinaciones lineales siguen siendo combinaciones lineales.
¿Cómo te salió?

Independencia lineal

Definición

Dependencia e independencia lineal

Enunciado formal Los vectores v₁, …, vk ∈ Rⁿ son linealmente independientes si la única solución de c₁·v₁ + c₂·v₂ + … + ck·vk = 0 es la trivial: c₁ = c₂ = … = ck = 0. Si existe alguna solución con algún cᵢ ≠ 0, son linealmente dependientes.
Por qué

La palabra clave es única: no basta con que la trivial funcione (siempre funciona). Independencia = la trivial es la única. Equivalente operativo: al poner los vectores en columnas y reducir, hay un pivote en cada columna (ninguna variable libre en el homogéneo).

Recall · enuncia de memoria

Da la definición formal de independencia lineal. Cuida decir «única solución trivial», no solo «la trivial funciona».

Respuesta
v₁, …, vk son independientes ⇔ c₁·v₁ + … + ck·vk = 0 obliga a c₁ = … = ck = 0 (la trivial es la única solución).
¿Cómo te salió?

Base y dimensión

Definición

Base de un subespacio

Enunciado formal Un conjunto B = {v₁, …, vk} ⊆ S es una base del subespacio S ⊆ Rⁿ si cumple las dos cosas:
  • 1B genera a S: gen(B) = S.
  • 2B es linealmente independiente.
Por qué

Generar sin independencia es sobrar vectores; ser independiente sin generar es faltar. La base es el punto justo: el mínimo conjunto que genera S. Consecuencia clave: todo vector de S se escribe de forma única como combinación lineal de la base.

Recall · enuncia de memoria

Define «base de S» con sus dos condiciones exactas.

Respuesta
B es base de S ⇔ (1) gen(B) = S y (2) B es linealmente independiente. Equivale a: todo vector de S se escribe de forma única en B.
¿Cómo te salió?
Teorema

La dimensión está bien definida

Enunciado formal Sea S ⊆ Rⁿ un subespacio. Todas las bases de S tienen el mismo número de vectores. Ese número común se llama dimensión de S y se denota dim(S).
Por qué

Es el teorema que legitima hablar de «la» dimensión: sin él, «dimensión» dependería de qué base elijas. Se apoya en el lema de reemplazo (Steinitz): un conjunto independiente nunca puede tener más vectores que uno generador. Como dos bases son ambas independientes y generadoras, se acotan mutuamente y quedan del mismo tamaño.

Recall · enuncia de memoria

¿Qué garantiza que dim(S) esté «bien definida»? Enúncialo.

Respuesta
Todas las bases de un mismo subespacio S tienen el mismo número de vectores; ese número es dim(S).
¿Cómo te salió?
Teorema

Dimensión vía pivotes (rango)

Enunciado formal Sean v₁, …, vk ∈ Rⁿ y A la matriz que los tiene por columnas. Entonces dim( gen{v₁, …, vk} ) = número de pivotes de la forma escalonada de A = rango(A). Además, las columnas de A con pivote forman una base de gen{v₁, …, vk}.
Por qué

Cada pivote marca una columna que no es combinación de las anteriores; las columnas sin pivote sí lo son (sobran). Por eso las columnas pivote son independientes y generan lo mismo: son una base, y su cantidad —el rango— es la dimensión.

Recall · enuncia de memoria

Relaciona dimensión del generado, pivotes y rango. ¿Qué columnas forman la base?

Respuesta
dim(gen de las columnas) = nº de pivotes = rango(A); las columnas originales con pivote forman una base.
¿Cómo te salió?

Espacio nulo, nulidad y homogéneos

Definición

Espacio nulo y nulidad

Enunciado formal Sea A una matriz m×n. El espacio nulo (o núcleo) de A es Nul(A) = { x ∈ Rⁿ : A·x = 0 } ⊆ Rⁿ. Es un subespacio de Rⁿ. Su dimensión se llama nulidad de A: nulidad(A) = dim( Nul(A) ).
Por qué

Nul(A) es el conjunto solución del sistema homogéneo A·x = 0; contiene siempre al 0 (por eso el homogéneo nunca es inconsistente), y si A·u = 0 y A·v = 0 entonces A·(u + v) = 0 y A·(c·u) = 0: cierra bajo suma y escalar, luego es subespacio.

Recall · enuncia de memoria

Define Nul(A) en notación de conjunto y di por qué es subespacio y qué es la nulidad.

Respuesta
Nul(A) = { x ∈ Rⁿ : A·x = 0 }; es subespacio (cierra bajo suma y escalar, contiene 0). nulidad(A) = dim(Nul(A)).
¿Cómo te salió?
Teorema

Rango + nulidad (teorema de la dimensión)

Enunciado formal Para toda matriz A de tamaño m×n: rango(A) + nulidad(A) = n (el número de columnas). Donde rango(A) = nº de columnas con pivote y nulidad(A) = nº de columnas libres (sin pivote).
Por qué

Al reducir A, cada columna cae en exactamente una de dos clases: con pivote (aporta al rango) o libre (aporta una variable libre, es decir un vector director del espacio nulo). Como cada columna es de una sola clase, pivotes + libres = n. De ahí la fórmula.

Recall · enuncia de memoria

Enuncia el teorema rango + nulidad, diciendo qué cuenta cada término y contra qué n se suma.

Respuesta
rango(A) + nulidad(A) = n (número de columnas); rango = columnas con pivote, nulidad = columnas libres.
¿Cómo te salió?
Método

Forma paramétrica del sistema homogéneo

Enunciado formal Al reducir A·x = 0, cada variable libre se toma como parámetro (t, s, …). Despejando las variables pivote, la solución general se escribe x = t₁·w₁ + t₂·w₂ + … + tp·wp, con p = nulidad(A). Los vectores directores w₁, …, wp forman una base de Nul(A).
Por qué

Un homogéneo siempre pasa por el origen (x = 0 lo resuelve), así que su solución es un subespacio puro, sin traslación. Los wi son independientes por construcción (cada uno lleva un 1 en «su» variable libre y 0 en las demás), y generan todas las soluciones: son base, y su cantidad es la nulidad.

Recall · enuncia de memoria

Describe cómo se obtiene la forma paramétrica de A·x = 0 y qué son los vectores director.

Respuesta
Cada variable libre = un parámetro; despejas pivotes; x = t₁·w₁ + … + tp·wp con p = nulidad. Los wi son una base de Nul(A).
¿Cómo te salió?

Los teoremas nucleares de Rⁿ

Teorema

n vectores independientes en Rⁿ son base

Enunciado formal Si v₁, …, vn son n vectores linealmente independientes en Rⁿ, entonces forman una base de Rⁿ. Simétricamente: si n vectores generan Rⁿ, entonces son independientes y forman base.
Por qué

En Rⁿ, dim = n. Un conjunto independiente de tamaño n ya alcanza el máximo de vectores independientes posibles, así que no puede faltarle nada por generar: genera todo Rⁿ. Es el atajo estrella de examen: con la dimensión correcta, basta probar una de las dos condiciones (independencia o generación), la otra sale gratis.

Recall · enuncia de memoria

Enuncia el teorema: ¿qué basta para que n vectores en Rⁿ sean base? Di las dos versiones.

Respuesta
n vectores independientes en Rⁿ ⇒ base. n vectores que generan Rⁿ ⇒ base. Con la dimensión justa, una condición implica la otra.
¿Cómo te salió?
Teorema

Cotas de dimensión y subespacios de Rⁿ

Enunciado formal Sea S ⊆ Rⁿ un subespacio. Entonces:
  • a0 ≤ dim(S) ≤ n.
  • bdim(S) = 0 ⇔ S = {0} (el subespacio trivial).
  • cdim(S) = n ⇔ S = Rⁿ (todo el espacio).
Y en general: cualquier conjunto de más de n vectores en Rⁿ es linealmente dependiente.
Por qué

Ningún conjunto independiente en Rⁿ puede superar n vectores (más columnas que renglones ⇒ hay variable libre ⇒ dependientes). Por eso la dimensión de cualquier subespacio está topada por n, y solo Rⁿ entero llega al tope.

Recall · enuncia de memoria

¿Entre qué valores está dim(S) para S ⊆ Rⁿ, y qué pasa con más de n vectores?

Respuesta
0 ≤ dim(S) ≤ n; dim = 0 ⇔ S = {0}; dim = n ⇔ S = Rⁿ. Más de n vectores en Rⁿ ⇒ siempre dependientes.
¿Cómo te salió?

La ficha de memoria

En el examen no hay formulario. Esto es lo que debes poder escribir de memoria, palabra por palabra.

  • ✓ Subespacio: (1) 0 ∈ S · (2) cerrado bajo suma · (3) cerrado bajo escalar.
  • ✓ gen{v₁,…,vk} = { c₁v₁ + … + ck vk : cᵢ ∈ R }; siempre es subespacio (el menor que los contiene).
  • ✓ Independientes ⇔ c₁v₁ + … + ck vk = 0 obliga a todos los cᵢ = 0 (trivial única).
  • ✓ Base = genera + independiente; todo vector se escribe único en la base.
  • ✓ Dimensión bien definida: todas las bases de S tienen el mismo tamaño = dim(S).
  • ✓ dim(gen de columnas) = nº de pivotes = rango; las columnas pivote son base.
  • ✓ Nul(A) = { x : A·x = 0 } es subespacio; nulidad(A) = dim(Nul(A)).
  • ✓ Teorema de la dimensión: rango(A) + nulidad(A) = n (nº de columnas).
  • ✓ Homogéneo: cada variable libre = parámetro; x = Σ tᵢ·wᵢ, los wᵢ son base de Nul(A).
  • ✓ n independientes en Rⁿ ⇒ base; n que generan Rⁿ ⇒ base (con dim justa, una implica la otra).
  • ✓ 0 ≤ dim(S) ≤ n; dim 0 ⇔ {0}, dim n ⇔ Rⁿ; más de n vectores ⇒ dependientes.
