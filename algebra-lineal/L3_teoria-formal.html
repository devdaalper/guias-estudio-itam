Qué es una transformación lineal

Definición

Transformación lineal

Enunciado formal Sean V y W espacios vectoriales sobre R. Una función T: V → W es una transformación lineal si para todo par de vectores u, v ∈ V y todo escalar c ∈ R cumple las dos condiciones:
  • 1Aditividad: T(u + v) = T(u) + T(v).
  • 2Homogeneidad: T(c·u) = c·T(u).
Equivalen a una sola: T(c·u + d·v) = c·T(u) + d·T(v) para todo c, d ∈ R.
Por qué

«Lineal» significa que T respeta las dos operaciones del espacio: da lo mismo sumar o escalar antes que después de aplicar T. En cada coordenada solo caben combinaciones de las variables por constantes: nada de x², xy, |x|, ni sumar un número fijo. Las dos condiciones juntas se resumen en que T preserva combinaciones lineales.

Recall · enuncia de memoria

Define transformación lineal T: V → W con sus dos condiciones exactas. Dilo antes de revelar.

Respuesta
T: V → W es lineal si (1) T(u + v) = T(u) + T(v) y (2) T(c·u) = c·T(u), para todo u, v ∈ V y todo c ∈ R. En una sola: T(c·u + d·v) = c·T(u) + d·T(v).
¿Cómo te salió?
Propiedad

El cero va al cero (filtro rápido)

Enunciado formal Si T: V → W es lineal, entonces T(0V) = 0W: el vector cero del dominio va al vector cero del codominio. Contrarrecíproco (filtro): si una función no manda el cero al cero, no es lineal, y se descarta sin calcular nada más.
Por qué

Sale de la homogeneidad con c = 0: T(0) = T(0·u) = 0·T(u) = 0. Es una condición necesaria pero no suficiente: que el cero quede quieto no garantiza linealidad, pero que se mueva sí la descarta. Por eso una regla como T(x, y) = (x + 1, y) muere al instante: manda (0, 0) a (1, 0).

Recall · enuncia de memoria

Enuncia qué le pasa al vector cero bajo una T lineal y cómo se usa como filtro. ¿Es condición suficiente?

Respuesta
T lineal ⇒ T(0) = 0 (sale de c = 0). Si una función mueve el cero, no es lineal (filtro). Pero es necesaria, no suficiente: que fije el cero no basta para ser lineal.
¿Cómo te salió?

La matriz canónica

Teorema

Toda T lineal es una matriz (matriz canónica)

Enunciado formal Toda transformación lineal T: Rⁿ → Rᵐ se escribe como T(x) = A·x para una única matriz A de tamaño m×n, la matriz canónica (o estándar) de T. Sus columnas son las imágenes de la base canónica: A = [ T(e₁) | T(e₂) | … | T(en) ], donde ej es el vector de la base canónica con un 1 en la posición j y ceros en el resto. La imagen de ej es la columna j de A.
Por qué

Cualquier x = x₁·e₁ + … + xn·en. Por linealidad, T(x) = x₁·T(e₁) + … + xn·T(en), que es exactamente A·x con esas columnas. La matriz es m×n: m renglones (dimensión del codominio) por n columnas (dimensión del dominio). Multiplicar por A reproduce toda la transformación de un golpe.

Recall · enuncia de memoria

¿Cómo se construye la matriz canónica de T: Rⁿ → Rᵐ y qué tamaño tiene? Di qué es cada columna.

Respuesta
A = [ T(e₁) | … | T(en) ]: cada columna j es la imagen del j-ésimo vector canónico. Es m×n (m = dim del codominio, n = dim del dominio) y cumple T(x) = A·x.
¿Cómo te salió?

Núcleo e imagen

Definición

Núcleo (kernel) y nulidad

Enunciado formal El núcleo de T: V → W es el conjunto de vectores del dominio que van al cero: ker(T) = { v ∈ V : T(v) = 0 }. Es un subespacio de V. Su dimensión es la nulidad: nulidad(T) = dim ker(T).
Por qué

Con la matriz A, ker(T) es el conjunto de soluciones de A·x = 0 (sistema homogéneo): por eso siempre contiene al menos el cero. Se calcula reduciendo A: las columnas sin pivote son las variables libres; cada una aporta un vector director del núcleo, así que la nulidad = número de columnas sin pivote.

Recall · enuncia de memoria

Define ker(T) y nulidad. ¿Con qué sistema se calcula y cómo se lee la nulidad al reducir?

Respuesta
ker(T) = { v : T(v) = 0 }, subespacio del dominio. Nulidad = dim ker(T). Se halla resolviendo A·x = 0; la nulidad = nº de columnas SIN pivote (variables libres).
¿Cómo te salió?
Definición

Imagen (rango como subespacio) y rango

Enunciado formal La imagen de T: V → W es el conjunto de vectores del codominio que sí se alcanzan: Im(T) = { T(v) : v ∈ V } = { w ∈ W : w = T(v) para algún v }. Es un subespacio de W, generado por las columnas de A. Su dimensión es el rango: rango(T) = dim Im(T).
Por qué

Como T(x) = A·x recorre todas las combinaciones de las columnas de A, la imagen es el espacio columna de A. El rango es el número de pivotes al reducir. Trampa clásica de examen: una base de la imagen son las columnas originales de A donde cayeron los pivotes, NO las columnas de la matriz ya reducida (reducir cambia las columnas).

Recall · enuncia de memoria

Define Im(T) y rango. ¿De qué matriz es el espacio columna y cuál es la trampa al dar su base?

Respuesta
Im(T) = { T(v) : v ∈ V }, subespacio del codominio = espacio columna de A. Rango = dim Im(T) = nº de pivotes. Base = columnas ORIGINALES con pivote (no las reducidas).
¿Cómo te salió?
Método

Hallar bases del núcleo y de la imagen

Enunciado formal Para T con matriz A, se reduce A a su forma escalonada reducida y se lee:
  • 1Rango = número de pivotes. Una base de Im(T) son las columnas de A original que quedaron con pivote.
  • 2Nulidad = número de columnas sin pivote. Para una base de ker(T), cada variable libre se pone como parámetro (z = t, …), se despejan las básicas y se agrupan los vectores director.
Por qué

Los pivotes separan las columnas en dos grupos que no se mezclan: con pivote → alimentan la imagen (rango); sin pivote → variables libres → alimentan el núcleo (nulidad). Escribir el núcleo como t·w₁ + s·w₂ + … expone directamente sus vectores director, que forman su base.

Recall · enuncia de memoria

Describe cómo, a partir de la matriz reducida, obtienes una base de la imagen y una base del núcleo.

Respuesta
Reduce A. Imagen: columnas ORIGINALES con pivote (rango = nº pivotes). Núcleo: pon las columnas sin pivote como parámetros libres, despeja las básicas y junta los vectores director (nulidad = nº columnas sin pivote).
¿Cómo te salió?

El teorema central

Teorema

Teorema de rango-nulidad (de la dimensión)

Enunciado formal Sea T: V → W lineal con V de dimensión finita n. Entonces rango(T) + nulidad(T) = dim V = n, es decir, dim Im(T) + dim ker(T) = número de columnas de A (la dimensión del dominio).
Por qué

Al reducir A, cada columna del dominio cae en exactamente un grupo: o tiene pivote (cuenta para el rango) o no lo tiene (cuenta para la nulidad). Ninguna cae en los dos ni se queda fuera, así que los dos conteos suman el total de columnas = dim del dominio. Ojo: el total es la dimensión del dominio, no del codominio. Si conoces dos de los tres números, el tercero sale solo.

Recall · enuncia de memoria

Enuncia el teorema de rango-nulidad con precisión. ¿La suma da la dimensión del dominio o del codominio?

Respuesta
Para T: V → W con dim V = n: rango(T) + nulidad(T) = dim V = n = número de columnas. La suma da la dimensión del DOMINIO, nunca la del codominio.
¿Cómo te salió?

Clasificar: inyectiva, suprayectiva, biyectiva

Teorema

Criterio de inyectividad (nulidad 0)

Enunciado formal T: V → W lineal es inyectiva (uno a uno) si y solo si su núcleo es trivial: T inyectiva ⇔ ker(T) = { 0 } ⇔ nulidad(T) = 0. Equivalente por rango-nulidad: rango(T) = dim V (todas las columnas son pivote).
Por qué

Como T es lineal, T(u) = T(v) ⇔ T(u − v) = 0 ⇔ u − v ∈ ker(T). Si el único elemento del núcleo es el 0, entonces u = v: es inyectiva. Basta con que algo distinto del cero vaya al cero para que pierda la inyectividad. Por eso el criterio se reduce a mirar la nulidad.

Recall · enuncia de memoria

Da el criterio de inyectividad en términos del núcleo y de la nulidad. ¿Por qué basta con el núcleo?

Respuesta
T inyectiva ⇔ ker(T) = { 0 } ⇔ nulidad = 0. Porque T(u) = T(v) ⇔ u − v ∈ ker(T); si el núcleo es solo el cero, u = v.
¿Cómo te salió?
Teorema

Criterio de suprayectividad y biyectividad

Enunciado formal T: V → W lineal es suprayectiva (sobre) si su imagen llena todo el codominio: T suprayectiva ⇔ Im(T) = W ⇔ rango(T) = dim W (dimensión del codominio). T es biyectiva (isomorfismo) ⇔ es inyectiva y suprayectiva a la vez ⇔ nulidad = 0 y rango = dim W.
Por qué

La imagen es un subespacio de W; llenarlo por completo equivale a que su dimensión (el rango) iguale a dim W. Cuidado: rango se compara con la dimensión del codominio (renglones de A), mientras la inyectividad se compara con la del dominio (columnas). La biyectividad exige ambas, y solo es posible si dim V = dim W.

Recall · enuncia de memoria

Enuncia los criterios de suprayectividad y de biyectividad con rango, imagen y dimensión del codominio.

Respuesta
Suprayectiva ⇔ Im(T) = W ⇔ rango = dim W (codominio). Biyectiva ⇔ inyectiva y suprayectiva ⇔ nulidad = 0 y rango = dim W (requiere dim V = dim W).
¿Cómo te salió?
Corolario

Reglas de dimensión (cotas rápidas)

Enunciado formal Para T: V → W lineal, comparando dim V (dominio) con dim W (codominio):
  • 1Si dim V > dim W, T nunca es inyectiva (nulidad ≥ dim V − dim W > 0).
  • 2Si dim V < dim W, T nunca es suprayectiva (rango ≤ dim V < dim W).
  • 3T puede ser biyectiva solo si dim V = dim W.
Por qué

Salen directo de rango-nulidad. El rango nunca supera el número de columnas (dim V) ni el de renglones (dim W). Si el dominio es más grande que el codominio, sobran columnas que se aplastan al cero (no inyectiva); si es más chico, no hay suficientes columnas para cubrir el destino (no suprayectiva). Estas cotas descartan casos sin calcular la matriz.

Recall · enuncia de memoria

Enuncia las tres reglas de dimensión que descartan inyectiva / suprayectiva / biyectiva sin calcular.

Respuesta
dim V > dim W → nunca inyectiva. dim V < dim W → nunca suprayectiva. Biyectiva solo si dim V = dim W. (El rango no supera ni columnas ni renglones.)
¿Cómo te salió?

Composición e inversa

Teorema

Matriz de la composición y de la inversa

Enunciado formal Si S y T son lineales con matrices canónicas A y B, la composición S ∘ T (aplicar T y luego S) también es lineal, y su matriz es el producto: matriz(S ∘ T) = A·B (en ese orden: la del segundo aplicado por la izquierda). T: Rⁿ → Rⁿ es invertible ⇔ es biyectiva ⇔ su matriz A es invertible (det A ≠ 0); entonces la matriz de T⁻¹ es A⁻¹.
Por qué

(S ∘ T)(x) = S(T(x)) = S(B·x) = A·(B·x) = (A·B)·x, así que la matriz es A·B. El orden importa porque el producto de matrices no conmuta. La inversa existe solo cuando T es biyectiva, y eso obliga a que A sea cuadrada y de determinante distinto de cero.

Recall · enuncia de memoria

¿Cuál es la matriz de S ∘ T y en qué orden? ¿Cuándo es invertible T y cuál es la matriz de T⁻¹?

Respuesta
matriz(S ∘ T) = A·B (S con matriz A, T con B; se aplica primero T). T es invertible ⇔ biyectiva ⇔ A invertible (det A ≠ 0); matriz de T⁻¹ = A⁻¹.
¿Cómo te salió?

La ficha de memoria

En el examen no hay formulario. Esto es lo que debes poder escribir de memoria, palabra por palabra.

  • ✓ Lineal ⇔ T(u + v) = T(u) + T(v) y T(c·u) = c·T(u). En una: T(c·u + d·v) = c·T(u) + d·T(v).
  • ✓ T lineal ⇒ T(0) = 0. Si mueve el cero, no es lineal (necesaria, no suficiente).
  • ✓ Matriz canónica: A = [ T(e₁) | … | T(en) ], tamaño m×n. T(x) = A·x.
  • ✓ Núcleo ker(T) = { v : T(v) = 0 } (subespacio del dominio). Nulidad = dim ker = columnas sin pivote.
  • ✓ Imagen Im(T) = espacio columna de A. Rango = dim Im = nº de pivotes. Base = columnas ORIGINALES con pivote.
  • ✓ Rango-nulidad: rango + nulidad = dim del dominio (nº de columnas). No del codominio.
  • ✓ Inyectiva ⇔ ker = { 0 } ⇔ nulidad 0. Suprayectiva ⇔ rango = dim del codominio.
  • ✓ Biyectiva ⇔ ambas ⇔ nulidad 0 y rango = dim W; requiere dim V = dim W.
  • ✓ dim V > dim W ⇒ no inyectiva · dim V < dim W ⇒ no suprayectiva.
  • ✓ matriz(S ∘ T) = A·B. Invertible ⇔ biyectiva ⇔ det A ≠ 0; matriz de T⁻¹ = A⁻¹.
